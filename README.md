# Employee Management System_V1

## Overview
Employee Management System is a secure backend application developed using Spring Boot. The application provides employee management functionalities with authentication, authorization, password encryption, OTP verification, password recovery, file upload, and profile image upload features.

## Technologies Used
* Java 21
* Spring Boot
* Spring Security
* Spring Data JPA
* JWT (JSON Web Token)
* BCrypt Password Encoder
* MySQL
* Maven
* Java Mail Sender

## Features

### Employee Management
* Create Employee
* Update Employee
* Delete Employee
* View Employee Details

### Authentication & Authorization
* JWT Authentication
* Spring Security Integration
* Role-Based Access Control (ADMIN / USER)

### Password Security
* BCrypt Password Encryption
* Secure Password Validation

### Forgot Password Module
* Email OTP Generation
* OTP Verification
* Password Reset Functionality

### File Management
* PDF/File Upload
* Profile Image Upload

### Validation & Exception Handling
* Request Validation
* Custom Exception Handling
* Global Exception Handling

## Project Structure

* Entity Layer
* Security Layer
* Exception Handling Layer
