# EmployeeManagementSystemV1
A secure Employee Management System developed using Spring Boot. Features include JWT Authentication, Spring Security, BCrypt Password Encryption, OTP-based Email Verification, Forgot Password functionality, File Upload, and Profile Image Management.

## Features
- Employee CRUD Operations
- JWT Authentication
- Spring Security Integration
- BCrypt Password Encryption
- Email OTP Verification
- Forgot Password & Reset Password
- File Upload
- Profile Image Upload
- Global Exception Handling
- Input Validation
